# Sentiment-Aware-Recommendation-System-for-OTT-Platforms
## Project Overview

The Sentiment-Aware Recommendation System for OTT Platforms is an AI-powered movie recommendation system that analyzes a user's emotions from text input and recommends movies based on their current mood.

Unlike traditional recommendation systems that only use viewing history or ratings, this system understands the user's emotional state using Natural Language Processing (NLP) and Deep Learning.

The project uses DistilBERT, a lightweight Transformer model, for emotion detection and combines it with a movie recommendation engine to provide personalized suggestions.

## Features

Emtion Detection from User Text
DistilBERT-based Deep Learning Model
Multi-Class Emotion Classification
Sentiment Analysis
Personalized Movie Recommendations
Genre-Based Recommendation
Fast Prediction
Interactive Streamlit Web Application
